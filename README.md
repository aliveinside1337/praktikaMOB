# ContactBook

Мобильное web-приложение **ContactBook** — личная адресная книга для управления контактами.

Проект реализован как **Web-приложение на FastAPI** с подключенной **SQLite БД**.

## Возможности

- Просмотр списка контактов
- Добавление нового контакта
- Редактирование контакта
- Удаление контакта
- Избранные контакты
- Категории контактов:
  - Семья
  - Друзья
  - Работа
  - Другое
- Фильтрация по категориям
- Поиск контактов
- Импорт/экспорт данных
  - JSON
  - CSV
  - vCard
- Экран настроек
- Хранение данных в SQLite

## База данных

В проекте используется SQLite база данных:

```text
contactbook.db
```

База создаётся автоматически при первом запуске приложения.

Создаются таблицы:

- `contacts` — контакты
- `settings` — настройки
- `profile` — профиль пользователя

## Установка и запуск

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Запустить приложение

```bash
python run_app.py
```

### 3. Открыть в браузере

```text
http://127.0.0.1:8000/
```

или:

```text
http://127.0.0.1:8000/index.html
```

## API

### Контакты

#### Получить все контакты

```http
GET /api/contacts
```

#### Создать контакт

```http
POST /api/contacts
```

Пример тела запроса:

```json
{
  "name": "Иван Иванов",
  "phone": "+7 (999) 000-00-00",
  "email": "ivan@example.com",
  "address": "Москва, ул. Ленина, 1",
  "category": "Работа",
  "favorite": true
}
```

#### Обновить контакт

```http
PUT /api/contacts/{contact_id}
```

#### Удалить контакт

```http
DELETE /api/contacts/{contact_id}
```

### Настройки

#### Получить настройки

```http
GET /api/settings
```

#### Обновить настройки

```http
PUT /api/settings
```

### Профиль

#### Получить профиль

```http
GET /api/profile
```

#### Обновить профиль

```http
PUT /api/profile
```

## Интерфейс

Интерфейс рассчитан на мобильный экран примерно **390x844**.

Основные экраны приложения:

- Главная
- Контакты
- Поиск
- Категории
- Данные / Импорт-Экспорт
- Новый контакт
- Редактирование контакта
- Настройки

## Экспорт данных

Приложение поддерживает экспорт контактов в форматы:

- JSON
- CSV
- vCard

CSV экспорт сделан с кодировкой, подходящей для открытия в русском Microsoft Excel.

## Импорт данных

На экране `Данные` можно импортировать контакты из JSON-файла.

## Примечание

Frontend использует `localStorage` для быстрого отображения данных в интерфейсе, но при запуске через FastAPI данные также синхронизируются с backend API и SQLite базой данных.
